# 图书类目自动标引系统

基于机器学习的图书自动分类系统，采用增强文本特征 + 手工数值特征 + 类别过滤策略，根据图书的标题、关键词、摘要及预处理特征自动判断中图法类目。

## 项目简介

本项目构建了一个图书自动标引系统，利用TF-IDF文本向量化与手工数值特征相结合的多维特征方案，对比朴素贝叶斯、逻辑回归与LinearSVC三种经典分类模型。通过类别过滤、增强文本构建和网格搜索调优，最终在25个高频类别上达到 **62.21%** 的测试集准确率。

### 系统能做什么

- 自动处理图书数据（12列CSV读取、文本增强、数值特征提取）
- 类别过滤（保留训练样本>=100的高频类别）
- 训练并对比三种经典模型（朴素贝叶斯、逻辑回归、LinearSVC）
- GridSearchCV自动调优超参数（C + class_weight）
- 生成6类可视化分析图表

## 特征工程

### 增强文本构建

利用数据集12列中的前5列（标题、关键词、摘要）+ 第6列（精简关键词）进行拼接，构建增强文本特征：

```
text = title + keywords + abstract + kw_simple
```

### 手工数值特征（5维）

| 特征 | 说明 |
|------|------|
| title_char_len | 标题字符长度 |
| abstract_char_len | 摘要字符长度 |
| abstract_word_count | 摘要词数 |
| keyword_count | 关键词数量 |
| has_abstract | 是否有摘要（0/1） |

### TF-IDF参数

```python
TfidfVectorizer(
    max_features=8000,     # 最多保留8000个特征
    ngram_range=(1, 2),     # 使用1-2词组合
    min_df=2,               # 至少在2个文档中出现
    max_df=0.95,            # 出现频率不超过95%的词保留
    sublinear_tf=True       # 对数变换，稳定数值
)
```

## 实验结论

1. **类别过滤有效**：过滤低频类别（<100样本）后，分类空间从264类压缩至25类，准确率大幅提升
2. **增强文本有增益**：利用精简关键词列补充文本信息，相比基础文本提升约5个百分点
3. **LinearSVC最优**：在所有指标上均表现最佳，调优后测试集准确率达62.21%
4. **class_weight=None更优**：网格搜索验证balanced策略在本数据集上反而降低准确率
